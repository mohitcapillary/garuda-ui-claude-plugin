**User Story 1:** As a marketer, I want to view all the tiers configured for the program along with their benefits

As a user, I want to view all the below details for all tiers in a program

**KPIs:**
- Total tiers
- No of active tiers
- No of deactivated/stopped tiers
- Total members

**Tier listing:**
- Filter by status:
  - Draft
  - Pending approval
  - Active
  - Deactivated/Stopped

**Tier details:**
- Tier Name
- Tier description
- Tier status
- No of members in the tier
- Eligibility criteria
- Upgrade condition
- Membership duration
- Upgrade schedule
- Nudges/communication
- Renewal criteria
  - Renewal condition
  - Renewal schedule
  - Renewal duration
  - Nudges/communication
- Downgrade criteria
  - Downgrade to which tier
  - Downgrade schedule
  - Nudges/communication/Expiry reminders
- Linked Tier Benefits
  - Show benefits linked to a tier grouped by category
